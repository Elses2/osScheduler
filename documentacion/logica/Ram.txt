Ram

Responsabilidad: Simula la memoria RAM principal del sistema.

    Atributos:
        static Ram instancia: Instancia única de Ram (Singleton).
        int tamano: Tamaño de la RAM.
        Pagina[] arrayMemoria: Array de páginas en RAM.

    Constructor: (Privado, Singleton)
        Ram(int tamanoTotal): Inicializa la RAM. Solo accedido desde getInstancia.

    Métodos:
        static Ram getInstancia(int tamanoTotal): Devuelve la instancia única de RAM.
        Pagina setPagina(int idL, Elemento elemento): Agrega una página.
        Pagina getPagina(int idL): Busca una página por dirección lógica.
        Pagina[] getArrayMemoria(): Devuelve el array de páginas.

    Patrón de diseño: Singleton
